# Security Policy

## Reporting a vulnerability

If you find a security issue, please **do not** open a public issue. Instead, send an email or use [GitHub Security Advisories](../../security/advisories/new).

## Important

- **Never** commit your `.env` file, cookies, tokens, or credentials
- The `.gitignore` excludes `.env`, cookies, browser data, and downloads
- Check your changes for passwords or tokens before every commit
